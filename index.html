Some Smells

Some smells are nice
like some flowers or some candles.
Some smells aren’t so nice
like driving past some monoculture farm on some highway.
Some smells make you happy
like the smell of some home-cooked meal in some home.
Some smells make you sad
like some burning smell from the fire eating up your local forest.
Some smells make you reminisce
like some smell of a loved one’s home.
Some smells make you frustrated
like the metallic smell in some dilapidated place.
Some smells make you hopeful
like some earthy smell in the garden.
Some smells make you want to grab the world by its shoulders and shake or punch or force some sense
into it and then cradle it and tell it that everything will be okay even though some smells tell you that
there isn’t much hope.
But you keep hoping anyway
because you have to.

(2) 11/13/2022 – The intersection of frustration and hope. An environmentalist starting to really like this writing stuff.
